# 🚢 Titanic Survival Prediction (Classification)

A machine learning project that predicts whether a passenger survived the Titanic disaster using classification techniques.

---

## 📌 Overview

This project uses the Titanic dataset to build a predictive model based on passenger features such as age, gender, class, and fare.

---

## 🎯 Objective

To train a classification model that determines survival probability using historical passenger data.
